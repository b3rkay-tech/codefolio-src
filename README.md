# Portfolio

A minimal developer portfolio built with React, Vite, and Tailwind CSS.  
Designed to be simple, fast, and easy to customize.

![License](https://img.shields.io/badge/license-Apache%202.0-blue.svg)
![React](https://img.shields.io/badge/built%20with-React-61dafb.svg)
![Tailwind CSS](https://img.shields.io/badge/styled%20with-Tailwind%20CSS-38bdf8.svg)

---

## Features

- Single configuration file for all personal data (`src/App.tsx`)
- Lightweight setup using React + Vite + Tailwind CSS
- Responsive design for all screen sizes
- Fast performance with optimized build output
- Easy to deploy on modern hosting platforms

---

## Sections

- Hero section with name, role, and links
- About section for personal description
- Projects section for showcasing work
- Skills section grouped by category
- Contact section with social links
